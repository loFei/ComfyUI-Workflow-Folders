<h2 id="chinese">中文</h2>

### ComfyUI Workflow Folders

    - ComfyUI Workflow Folders 旨在增强 ComfyUI 原生的工作流面板。 
    提供目录管理功能和右键菜单，在 ComfyUI 中快速完成工作流分类管理。

### ✨核心功能

    - 📂 智能层级操作：根据当前右键点击的目录层级。
    精准执行 添加文件夹、删除、重命名 及 拷贝 操作，实现灵活的工作流分类整理。

### ⚠️使用注意

    - ⚠️ 重要提示：本项目仅针对目录结构进行操作，未覆盖原生的工作流文件右键菜单，
    因此，使用前请确保您已在 ComfyUI 中保存过至少一个工作流，以便插件加载并显示工作流面板及目录结构。

### 安装方式

    ```
    cd ComfyUI/custom_nodes
    git clone https://github.com/loFei/ComfyUI-Workflow-Folders.git
    ```

<h2 id="english">English</h2>

### ComfyUI Workflow Folders

    - ComfyUI Workflow Folders is designed to enhance the native ComfyUI workflow panel
    by providing directory management and context menus, enabling quick workflow classification
    within ComfyUI.

### ✨ Core Features

    - 📂 Smart Hierarchical Operations: Precisely execute Add Folder, Delete, Rename,
    and Copy actions based on the hierarchy level of the directory you right-clicked,
    allowing for flexible workflow organization.

### ⚠️ Usage Notes

    - ⚠️ Important: This project focuses solely on directory structure operations and
    does not override the native workflow file context menu. Therefore, please ensure
    you have saved at least one workflow in ComfyUI before using this plugin, so that
    the workflow panel and directory structure are loaded correctly.

### Installation

    ```
    cd ComfyUI/custom_nodes
    git clone https://github.com/loFei/ComfyUI-Workflow-Folders.git

    ```
